# 🎁 CrowdGifting
> 여러 사람이 함께 선물을 준비하는 크라우드 펀딩 기반 선물 서비스
> 🦁 성신여자대학교 멋쟁이사자처럼 13기 데모데이 프로젝트

## 📌 프로젝트 소개
**CrowdGifting**는  
하나의 선물을 위해 여러 사용자가 펀딩에 참여하여  
목표 금액을 달성하면 선물을 구매·정산할 수 있는  
**크라우드 기반 선물 서비스**입니다.

- 친구, 가족, 동료와 함께 선물을 준비할 수 있습니다.
- 펀딩 진행률에 따라 선물 이미지가 점점 선명해지는 UI를 제공합니다.
- 펀딩 달성 시 정산 기능을 통해 선물 구매가 가능합니다.

## 🛠 주요 기능
- 펀딩 생성 및 참여
- PortOne 결제 연동
- 펀딩 달성률 기반 UI 변화 (이미지 블러 해제)
- 펀딩 달성 시 정산 처리
